# **App Name**: EchoScript

## Core Features:

- Main Typing Panel: A central, large text input area that behaves like a message composer for text entry and display.
- On-screen A-Z Keyboard: A virtual keyboard layout with multiple rows of A-Z characters for direct input into the typing panel.
- Voice Typing Activation: A microphone button that, when activated, displays a 'Listening...' status to simulate voice input readiness.
- New Word Suggestion Interface: A plus sign button to trigger a modal or section where users can propose new words or phrases for the language.
- Educational Impact Section: A dedicated area to explain the importance and impact of the application for language preservation.
- Prototype Badge Display: A small, clear 'Prototype' badge integrated into the interface to indicate its development stage.

## Style Guidelines:

- Primary brand color: A deep, professional blue (#1F66AD) symbolizing academic depth and technological innovation, providing a strong anchor for text and interactive elements.
- Background color: A very subtle cool light grey (#F0F2F5) providing a clean, spacious canvas that complements the primary blue without distraction, adhering to a dark-on-light aesthetic.
- Accent color: A vibrant turquoise/cyan (#33CCCC) for interactive elements and highlights, introducing a modern, energetic feel for calls to action and important indicators.
- All text (headlines and body) will use the 'Inter' sans-serif font for its modern, highly legible, and neutral aesthetic, supporting a strong, clean academic-tech style.
- Use tasteful and universally recognizable icons for core functionalities such as keyboard, microphone, speaker, dictionary (conceptual), and add/plus actions, maintaining a consistent minimalist line-art style.
- The layout features a prominent header with the app title and subtitle, a centrally located main typing panel, and clear divisions for keyboard and supplementary features, all designed with soft cards, rounded corners, and subtle shadows for a polished prototype feel.
- The application will be built as a single-page app with a responsive design, optimizing for clear presentation on laptop screens suitable for hackathon demonstrations.
